# Deploy Collabora in Nextcloud

* Step 1: Install docker
    ```
    https://docs.docker.com/engine/install/ubuntu/
    ```
* Step 2: Install docker compose
    ```
    sudo apt-get install docker-compose
    ```
* In current folder run 
    ```
    docker-compose up -d
    ```
* Nextcloud setup
    ```
    goto http://localhost
    ```
